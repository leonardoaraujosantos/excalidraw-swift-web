# @cyberdynecorp/excalidraw-relay

WebSocket relay server for collaboration — rooms, presence, and version-reconciled scene sync.

Part of the **[Excalidraw — Native Swift + TypeScript/Svelte port](https://github.com/CyberdyneCorp/excalidraw-native)**: a from-scratch TypeScript + Svelte 5 implementation of Excalidraw whose packages share a data model, `.excalidraw` file format, and a custom WebSocket collaboration protocol with a native Swift/SwiftUI twin.

## Install

```sh
npm install @cyberdynecorp/excalidraw-relay
```

ESM-only. See the [monorepo README](https://github.com/CyberdyneCorp/excalidraw-native#readme) for the full package list and usage.
